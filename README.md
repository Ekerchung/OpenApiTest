# OpenApiTest

臺灣期貨交易所API串接測試、實作增刪改查功能

## 使用技術

- 前端
  - html
  - css
  - javascript
  - ajax
- 後端
  - SpringBoot Data Jpa
  - Thymeleaf
- 資料庫
  - MySql
- 其他
  - Git、Github
  - Maven
  - MVC設計模式
  - RESTful API
  - Tomcat
